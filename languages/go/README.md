# Go Language Demos

This directory contains demos and projects written in Go.

## Projects

*   **[gcuk-2025](./gcuk-2025)**: Demo code for "Building MCP Servers in Go" at GopherCon UK 2025.
*   **[golab2025](./golab2025)**: Demo code for "Building MCP Servers in Go" at GoLab 2025.
*   **[gopherconsa2025](./gopherconsa2025)**: Demo code for GopherCon South Africa 2025.
*   **[pacman](./pacman)**: A complete Pac-Man clone using Ebitengine.
*   **[testquery](./testquery)**: A tool to query Go test results using SQL.
