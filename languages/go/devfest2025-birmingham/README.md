# DevFest Birmingham 2025

Resources and materials for DevFest Birmingham 2025.

**Official Website:** [DevFest Birmingham 2025](https://gdg.community.dev/events/details/google-gdg-birmingham-presents-devfest-birmingham-2025/)

**Slides:** [A Gopher's Guide to Vibe Coding](https://speakerdeck.com/danicat/a-gophers-guide-to-vibe-coding-b7ee3039-5fc6-45a0-8e94-ef4ade3f1e73)

## Contents

-   **[geminicli](./geminicli)**: A demonstration of the Gemini CLI's capabilities, including code generation and multi-modal understanding.
-   **[stitch](./stitch)**: Supporting files for the "Stitch by Google" demo, focused on UI redesign.
-   **[agy](./agy)**: A Go-based project featuring "Pacman" and "Selene" demos.
