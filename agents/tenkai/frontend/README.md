# Tenkai Dashboard

The web-based control plane and analysis dashboard for the Tenkai agent experimentation framework.

## Features

- **Real-time Monitoring:** Watch experiment progress, active runs, and live logs.
- **Detailed Investigation:** 
    - Inspect full conversation traces with collapsible event cards.
    - View tool usage, arguments, and outputs (JSON beautified).
    - Analyze failures with dedicated status banners and error logs.
- **Statistical Analysis:** 
    - Comparative performance tables with significance testing (p-values).
    - Executive summaries generated by AI (if configured).
    - Track metrics like Success Rate, Duration, Token Usage, and Failed Tool Calls.
- **Configuration:** View and audit the exact configuration (YAML) used for any experiment.

## Development

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```
    This starts the frontend at [http://localhost:3000](http://localhost:3000).
    
    > **Note:** The frontend requires the Tenkai backend API to be running (`tenkai --serve`).

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Components:** Radix UI (via shadcn/ui patterns)
- **Icons:** Lucide React
- **Data Fetching:** Native `fetch` with polling for live updates.
